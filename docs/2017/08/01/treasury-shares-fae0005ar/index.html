Question

In 2017 Apple has continued its policy of buying back shares. Which of the following is correct:

  1. Apple might buy back shares so that it can increase its voting rights
  2. Apple might buy back shares so it can offer stock option compensation to its employees
  3. Apple might buy back shares so it can decrease its Earnings-per-Share
  4. Apple might buy back shares so it can decrease its par value

Solution:

Point 2.
